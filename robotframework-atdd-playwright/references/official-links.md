# Hivatalos Hivatkozasok

Az uj skill minden esetben ezekhez igazodjon, es ha kerdeses a megoldas, ezek legyenek az elsodleges forrasok.

## Robot Framework

- Robot Framework: https://robotframework.org/
- Robot Framework User Guide: https://robotframework.org/robotframework/latest/RobotFrameworkUserGuide.html
- Robot Framework Guides: https://docs.robotframework.org/

## Browser Library

- Robot Framework Browser library: https://docs.robotframework.org/docs/different_libraries/browser
- Browser library repository: https://github.com/MarketSquare/robotframework-browser

## Playwright

- Playwright documentation: https://playwright.dev/

## Alkalmazasi Szabaly

- Ha a user kerese es a bevett gyakorlat elter, a hivatalos Robot Framework dokumentacio az elsodleges kiindulopont.
- A skill ne talaljon ki sajat, a dokumentacioval ellentetes mintakat.
- A megoldas legyen olvashato, karbantarthato, es kozel alljon a Robot Framework ajanlott hasznalatahoz.
